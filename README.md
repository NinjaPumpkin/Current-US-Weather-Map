# US Weather Map

Real-time US weather, satellite, air quality, fire, and disaster data dashboard.

## Data Sources

| Source | API | Key Required | Description |
|--------|-----|:---:|-------------|
| Open-Meteo | REST | No | Weather forecasts, current conditions |
| GOES-16/18/19 | NOAA | No | Satellite imagery (CONUS, Full Disk) |
| AirNow | EPA | Yes | AQI data from EPA monitors |
| AQICN/WAQI | REST | Yes | Global AQI from 130+ countries |
| OpenWeather | REST | Yes | Air pollution and AQI |
| NASA FIRMS | REST | Yes | Active fire hotspots (MODIS/VIIRS) |
| NASA EONET | REST | No | Natural disaster events |

## Setup

1. Clone the repo
2. Open index.html in a browser
3. Set API keys in browser localStorage:
   - AIRNOW_API_KEY - get from https://docs.airnowapi.org/
   - AQICN_TOKEN - get from https://aqicn.org/api/
   - OPENWEATHER_KEY - get from https://openweathermap.org/api
   - FIRMS_MAP_KEY - get from https://firms.modaps.eosdis.nasa.gov/api/

## GitHub Actions

Satellite and disaster data auto-fetch every 6 hours via GitHub Actions.
Results saved to data/ directory.

## Architecture

js/
  data-loader.js              Consolidated loader (imports all data-loaders)
  data-loaders/
    satellite-data.js          GOES-16/18/19 viewer URLs
    weather-data.js            Open-Meteo weather
    airnow-data.js             AirNow.gov AQI
    aqicn-data.js              WAQI/AQICN AQI
    openweather-airpollution-data.js  OpenWeather air pollution
    waqi-data.js               WAQI heatmap/station data
    firms-fire-data.js         NASA FIRMS fire hotspots
    eonet-disaster-data.js     NASA EONET disaster events
scripts/
    fetch-satellite-metadata.js  CI script: fetch GOES metadata
    fetch-satellite-imagery.js   CI script: download satellite images
data/                          Auto-generated by CI
.github/workflows/
    fetch-satellite.yml        Runs every 6 hours
